安装脚本

Chrome 浏览器需要安装 TamperMonkey 扩展程序 来运行用户脚本。
如果您已经安装了上述扩展程序，您可以点此安装脚本，并根据扩展程序的提示确认安装。
Firefox 浏览器需要安装 GreaseMonkey 附加组件 来运行用户脚本。
如果您已经安装了上述附加组件，您可以点此安装脚本，并根据附加组件的提示确认安装。
Opera 浏览器需要安装 TamperMonkey Beta 扩展 来运行用户脚本。
如果您已经安装了上述扩展，您可以点此安装脚本，并根据扩展的提示确认安装。
Safari 浏览器需要安装 TamperMonkey 扩展 来运行用户脚本。
如果您已经安装了上述扩展，您可以点此安装脚本，并根据扩展的提示确认安装。
